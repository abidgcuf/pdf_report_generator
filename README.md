# 📊 Student Result Automation System (Python + Streamlit)

A real-world **Excel automation web app** that processes student data, generates grades, calculates performance, and provides downloadable reports.

This project is built using **Python, Streamlit, and Pandas** and is deployed as a live web application.

---

## 🚀 Features

- 📂 Upload Excel file (.xlsx)
- 🧮 Automatic calculations:
  - Total Marks
  - Percentage
  - Grades
- 🏆 Topper detection
- 📊 Interactive dashboard
- 📈 Data visualization
- 📄 Generate PDF reports
- ⬇️ Download results (CSV / ZIP option)

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🌐
- Pandas 📊
- OpenPyXL 📂
- ReportLab 📄
